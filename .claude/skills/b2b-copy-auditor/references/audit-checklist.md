# B2B Copy Audit Checklist — Per-Page Scoring

Use this checklist when auditing a single page. Score each item 1-4:
1 = Missing / Failing | 2 = Weak / Partial | 3 = Adequate | 4 = Strong

| # | Criterion | Score | Notes |

### Interpretation

- **127-156 (81-100%):** Strong. Minor refinements only.
- **95-126 (61-80%):** Needs targeted improvements. Identify the weakest sections
  and prioritize rewrites there.
- **64-94 (41-60%):** Significant gaps. Full rewrite of weak sections recommended.
  The page is likely underperforming.
- **Below 64 (<41%):** Fundamental restructure needed. The page is probably not
  converting and may be actively repelling the target audience.

---

## Per-Item Rewrite Trigger

Any individual item scoring 1 or 2 should generate a rewrite suggestion. The
suggestion should include:

1. **What's wrong** — Name the specific failure.
2. **Why it matters** — Connect to the principle (Bly or Wiebe) that it violates.
3. **Suggested rewrite** — Provide alternate copy. Not a direction. Actual words.
4. **Confidence level** — How confident is the suggestion without VOC data?
   (High = structural/clarity issue. Medium = messaging issue where VOC would
   help. Low = tone/voice issue that depends on brand positioning.)
